### 如何添加数据库密码:
seafile自动安装mariadb(是mysql的开源版本),里面会有mysql数据库,mysql中包含一张user的table
可以使用:
```
sudo mysql -u root mysql
>use mysql;
>show tables;
```
更新密码:  
update user set password="openthos" where host="localhost";
